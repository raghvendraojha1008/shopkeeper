Calls nativePdfService.generateAndShare() which on Android uses the native PdfGenerator Capacitor plugin → writes to cache → opens the system share sheet (WhatsApp, Drive, Gmail, etc.)
Shows "PDF ready to share" on success, "Failed to generate PDF" on failure
statement.ts — same fix applied: removed all jsPDF/jspdf-autotable static imports, rewrote to build sections and call nativePdfService.generateAndShare(). Still correctly applies the pre-period opening balance runningBalance and includes misc charges in the ledger timeline.
