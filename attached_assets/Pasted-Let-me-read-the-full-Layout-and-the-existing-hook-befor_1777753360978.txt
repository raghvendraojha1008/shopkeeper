New hook — useSidebarBadges.ts

Makes exactly 2 Firestore queries on mount and every 5 minutes (same interval as useExpiringCount):

Fetches the 200 most recent subscription_logs entries and counts:
Events with timestamp >= 24 h ago → Audit Log badge
Events with action === 'grant' and timestamp >= 7 days ago → Giveaway Log badge
Fetches announcements filtered by isActive === true → Announcements badge
Uses Promise.allSettled so a rules failure on one collection doesn't kill the other, and failures are silent (badges are non-critical UI).

Updated Layout — badge system

Sidebar item	Badge	Colour	Meaning
Users	subscriptions expiring ≤ 3 days	orange	Urgent — act soon
Giveaway Log	grants in last 7 days	violet	Recent activity
Announcements	currently active announcements	green	Status — things are live
Audit Log	new events in last 24 h	blue	New activity to review
Each badge has a title tooltip explaining what the number means. Badges only appear when count > 0, capped at 99+. The NavBadge component is typed with a BadgeColor union so new colours can be added easily.
